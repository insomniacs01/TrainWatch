# GitHub Launch Metadata

## Recommended repo name

**`train-watch`**

原因：

- 简短，好记
- 读起来像一个产品名，而不是脚本名
- 同时兼容现在的 `v1` 和未来的 `v2`

发 GitHub 时，建议你明确写成：

- `v1`: self-hosted mobile PWA, already usable
